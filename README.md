Memes storage
---
<b>Service for storage your favorite memes.</b></br>

```
Technologies:
* FastApi
* React
* S3 storage - Minio
* Docker
* Poetry
* Pytest
```

---
To start localy via docker:
```commandline
git clone git@github.com:al-ov73/mems-storage.git && cd mems-storage
```
fill **.env-non-dev**
```
make docker
```
